A Secretária de Saúde de Feira de Santana pretende instalar um sistema de
cadastro por toda a cidade, para que possa coletar dados das diversas crianças do
Município durante o ano corrente. Com isso podendo estipular quais atendimentos de
saúde precisam ser ampliados na cidade.
O problema proposto tem como objetivo auxiliar os Agentes Comunitários de
Saúde (ACS) no seu trabalho diário, inicialmente coletando dados individuais das
crianças do bairro Feira VI, onde pretende-se utilizar um sistema de cadastro que será
instalado no Tablet de cada agente. Sendo que, esse programa irá registrar de cada
criança informações como: data de nascimento, peso, sexo, se tem plano de saúde e,
além disso, informar se a criança tem problema cardíaco, microcefalia e diabetes.
Tal questão precisará de um linguagem de programação para implementar o
sistema. Por critérios didáticos, será utilizada a linguagem C, que é uma linguagem de
programação estruturada de fácil aprendizagem e compreensão. Primeiramente haverá
um protótipo desenvolvido para Desktop (compatível com o Sistema Operacional
Windows) de maneira que possa avaliar o aplicativo antes de ir para a plataforma de
dispositivos móveis.

Tópicos: resumo do problema, descrição breve da solução
2. Metodologia
Ao receber o problema no dia 21 de setembro, foi discutido em grupo quais
impasses poderiam surgir a partir do problema, entre eles, a questão se seria necessário
ter um Sistema de Gerenciamento de Banco de Dados (SGBD), tal pergunta foi
prolongada durante algumas sessões, mas ao analisar devidamente o problema, foi
entendido que não há necessidade de ter um SGBD; ao invés disso, o programa ao
finalizar cada cadastro com sucesso, será acrescentado um valor para determinada
variável.
Para o desenvolvimento do aplicativo foi de grande importância utilizar as
estruturas de condição e repetição, visto que para o cadastro é necessário criar
condicionais caso a informação obtida seja verdadeira ou não e, além disso, há à
estrutura de repetição, que foi utilizada para incrementar os valores já salvos nos
cadastros anteriores.
